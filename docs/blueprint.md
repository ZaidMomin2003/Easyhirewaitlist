# **App Name**: Charon Waitlist

## Core Features:

- Central Form: Collect user's Name, Business, and Email in a central form built with ShadCN components.
- Submission Button: A clear, prominent submission button to submit the form data.
- Immediate Confirmation: Provide immediate visual feedback upon successful form submission.
- Data Storage: Integrate with Firestore for storing the collected data.

## Style Guidelines:

- Background color: Deep charcoal (#222222) to set a sophisticated, dark tone.
- Primary color: Deep indigo (#483D8B) for a modern, luxurious feel, contrasting subtly against the dark background.
- Accent color: Subtle neon green (#39FF14) for highlights and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern aesthetic.
- Single, no-scroll section layout for a focused user experience.
- Subtle fade-in effects and transitions on form elements.
